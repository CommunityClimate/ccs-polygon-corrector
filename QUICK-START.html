🎯 How to Run (Choose One)

Option 1: VS Code (Easiest)

  1. Open the field-polygon-app folder in VS Code
  2. Install "Live Server" extension
  3. Right-click index.html → "Open with Live Server"
  4. Browser opens automatically! 🎉

Option 2: Python (Quick)

  1. Open terminal/command prompt in the field-polygon-app folder
  2. Run this command:
python -m http.server 8000
  1. Open browser: http://localhost:8000

Option 3: Node.js

  1. In the field-polygon-app folder:
npx http-server -p 8000
  1. Open browser: http://localhost:8000
⚠️ Important: Don't just double-click index.html!
Modern browsers block ES6 modules over file:// protocol.
You MUST use a local web server (any option above).

🧪 Test It Out

Quick Test with Sample Data

  1. Start the app (using one of the methods above)
  2. Click "Choose File" under "Import Data"
  3. Select sample-data.geojson
  4. Select a field from the dropdown
  5. Click "Load Field"
  6. Click "Validate Polygon"
  7. Try "Auto-Correct" on invalid fields!

❓ Troubleshooting

Map not loading?
→ Check internet connection (map tiles need internet)

"Failed to load module" error?
→ Use a local web server, don't open file directly

Import not working?
→ Check file format matches sample-data.geojson structure

Data disappeared?
→ Avoid private/incognito mode (uses LocalStorage)
→ Export data regularly as backup
